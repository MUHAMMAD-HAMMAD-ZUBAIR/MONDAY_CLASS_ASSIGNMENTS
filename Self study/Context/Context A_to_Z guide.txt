## 4ï¸âƒ£ ğŸ Python Example (Beginner Friendly)

```python
# Simple in-memory context
history = []

def chat(user_text):
    history.append({"role": "user", "content": user_text})
    # AI uses 'history' to reply
    reply = f"(AI replies based on {len(history)} messages)"
    history.append({"role": "assistant", "content": reply})
    print(reply)

# Usage
chat("Hey, I'm Hammad.")
chat("I love pizza.")
chat("What do you know about me?")
```
